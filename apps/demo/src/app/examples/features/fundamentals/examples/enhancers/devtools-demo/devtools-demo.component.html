📖 What This Demo Shows

This demo showcases SignalTree's DevTools integration for debugging and monitoring your application state. The Redux DevTools extension auto-connects, tracks path-based actions, and supports time-travel jumps while SignalTree keeps a live view of state and metrics.

🎯 How to Use This Demo:

  1. Modify State: Use the controls on the left to change counter, user info, and todos
  2. Inspect in Redux DevTools: Open the Redux DevTools browser extension to see path-based actions and jump between states
  3. Take Snapshots: Save current state and restore it later
  4. Review History: Inspect path-based actions in Redux DevTools and time-travel to prior states
  5. Inspect State: View live state data at the bottom

✨ Key Features:

Time-Travel Debugging State Snapshots Path-Based Actions Action Metadata Redux DevTools Integration Live State Inspection

🔌 Redux DevTools

This tree is connected to the Redux DevTools extension (when installed) with enableBrowserDevTools: true. Open the extension and pick "DevTools Demo" to watch path-based actions stream in as you change state, and to time-travel between states.

Detailed performance counters (per-module update counts, timings, cache stats) are collected internally by the enhancer and surfaced through the extension in dev builds — they are not re-rendered in this panel.
