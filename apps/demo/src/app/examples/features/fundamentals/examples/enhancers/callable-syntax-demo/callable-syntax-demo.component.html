✨ Comprehensive Callable Syntax with @signaltree/callable-syntax

This demo showcases the complete callable syntax that transforms at build time:

  • tree.$.name('value') → tree.$.name.set('value')
  • tree.$.age(n => n + 1) → tree.$.age.update(n => n + 1)
  • tree.$.profile.email('new@email.com') → tree.$.profile.email.set('new@email.com')
  • tree.$.items(arr => [...arr, item]) → tree.$.items.update(arr => [...arr, item])

Note: TypeScript may show errors in the IDE - that's expected! The @signaltree/callable-syntax transform fixes them at build time.
