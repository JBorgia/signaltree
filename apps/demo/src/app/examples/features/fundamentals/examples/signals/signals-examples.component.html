Signals — Fundamentals

Minimal, focused examples demonstrating signal() basics and UI reactivity.

Basic Counter

The smallest possible signal + updater pattern.

#signals#state-management#basic
{{ count() }} 
const count = signal(0);
count.update(v => v + 1);

Reactive Form Control

Bind an input to a signal and derive a greeting.

#signals#forms
{{ greeting() }} 
const name = signal('');
const greeting = computed(() => `Hello, ${name().trim() || 'friend'}!`);
