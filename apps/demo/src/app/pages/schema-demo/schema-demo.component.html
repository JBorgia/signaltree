@signaltree/schema

Schema-driven validation against tree leaf paths. StandardSchema-compatible (Zod, Valibot, ArkType, …). Async-first. Observe-only — never blocks writes.

Live form

Schemas are registered at name, email, age, and username. The username field uses an async Zod refine that simulates a 300ms server check (try typing admin, root, or system to see the async path).
