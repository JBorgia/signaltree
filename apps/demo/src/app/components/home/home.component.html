Reactive JSON for Angular

Model nested state as data — not reducers, selectors, and ceremony.

SignalTree gives Angular apps a path-based, type-safe state model that stays readable as your data gets deeper and your app gets bigger.

Reactive JSON means your state still looks like data. Keep one runtime tree, compose typed slices, and add DevTools, time travel, persistence, forms, realtime, and events when you need them.

SignalTree is not @ngrx/signals. Different library, different package (@signaltree/core) — no withState/withMethods wrappers, no actions, no reducers. Your state literal is the API: read and write any path directly, at any depth.

Use SignalTree if you need

  • Optimistic UI with rollback (updateAndReport)
  • Undo / redo (timeTravel enhancer)
  • Typed normalized collections with O(1) lookups (entityMap)
  • Reactive forms with validation, wizards, and persistence (form() marker)
  • localStorage autosave with migrations (stored() marker)
  • State that mirrors your data shape, not Redux ceremony
